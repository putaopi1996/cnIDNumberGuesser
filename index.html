GuessID

    Introduction

    This is a project designed to guess and complete Chinese Mainland Resident Identity Card (ID Card) numbers. If you can't remember a part of your ID card number, or if some digits are missing, you can use this tool to complete the ID number.
    This tool provides both Python and Web versions, and you are currently using the Web version.
    Note: This project is for academic research purposes only. Please comply with local laws and regulations and do not use it for illegal purposes. The author is not responsible for any consequences if you deviate from this path.

    Visit this site to start using it. Similarly, enter an 18-digit ID card number, replacing the missing digits with asterisks "*".

    Click "GuessID" to run. After obtaining the results, they will be displayed below. You can also click "Download" to save the results in 'guessedIDs.txt' and download them to your local machine.

    Project source code: https://github.com/Gloridust/cnIDNumberGuesser
